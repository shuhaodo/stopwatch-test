## Usage

Simply open `index.html` in any modern web browser:

1. Click **Start** to begin timing
2. Click **Stop** to pause
3. Click **Start** again to resume
4. Click **Reset** to clear the timer back to 0.00

## Technical Details

- **Single HTML File**: All code (HTML, CSS, JS) in one file
- **High-Resolution Timing**: Uses `performance.now()` for accuracy
- **Smooth Updates**: Uses `requestAnimationFrame` for 60fps display
- **Browser Compatibility**: Works in Chrome, Firefox, Safari, and Edge
